## Project Title
我的餐廳清單
## Installation
1. Git clone
```
git clone https://github.com/BAGEBEAR/2-3_restaurant_list.git
```
2. 安裝npm
```
npm install
```
3. 

3. 輸入執行指令，開啟本地伺服器
```
npm run dev
```
## Features
### 列出我的餐廳清單
1. 使用者可以在首頁看到餐廳與它們的簡單資料
2. 使用者可以再點進去看餐廳的詳細資訊：類別、地址、電話、描述、圖片
3. 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
4. 使用者可以透過搜尋餐廳類別來找到特定的餐廳
